Build a modern professional web application called:

COST ANALYSIS PER CONTAINER

This is an enterprise-style container clearing cost analysis and operations management web application used by a port clearing/logistics company.

The system tracks all operational and financial costs related to clearing containers from the port.

The application must allow staff to input operational charges while automatically calculating total container cost, clearing charges, and gross profit.

The interface must be modern, clean, and professional, similar to premium accounting or logistics management software.

Use dashboards, charts, smart tables, and collapsible sections to make the interface easy to understand.

SMART CALCULATIONS

Total Cost =
Sum of all numeric expense fields.

Gross Profit =
Clearing Charges minus Total Cost.

All numeric fields must default to 0.

Numbers must display with comma formatting.

Staff cannot manually edit Total Cost or Gross Profit.
